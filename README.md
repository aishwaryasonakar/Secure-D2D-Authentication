# Secure Device-to-Device Authentication Without Central Authority Using ECC, ECDH, and AES-128.

## Project Description

This project simulates secure communication between two virtual devices without using cloud server.

## Features

- Secure Device-to-Device Authentication
- ECC (Elliptic Curve Cryptography) for Public/Private Key Generation
- ECDH (Elliptic Curve Diffie-Hellman) Shared Secret Key Exchange
- AES-128 Secure Message Encryption and Decryption
- Challenge-Response Authentication Mechanism
- Replay Attack Prevention using Nonce Verification
- Secure Communication between Virtual Devices
- Authentication and Communication Event Logging
- Python-based Simulation

## Technologies Used

- Python
- ECC (Elliptic Curve Cryptography)
- ECDH (Elliptic Curve Diffie-Hellman)
- AES-128
- Cryptography Library
- VS Code
